# Starbucks Rewards - Practica Frontend

Replica desktop de la pagina Starbucks Rewards propuesta en Frontend Practice.

## Integrantes

- Tomas

## Resolucion de pantalla usada

- 1280 x 800 px

## Archivos principales

- `index.html`: estructura semantica de la pagina.
- `styles.css`: estilos desktop, header sticky, tabs y secciones.
- `starbucks-assets/`: recursos visuales incluidos en la entrega.

## Recursos consultados

- Frontend Practice: https://www.frontendpractice.com/projects/starbucks
- Assets locales: `starbucks.zip`

## Nota

Proyecto realizado con fines educativos. No es una pagina oficial de Starbucks.
